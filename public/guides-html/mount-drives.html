Drives

Mounting

Create directory, add permissions Then run, changing partition and mount point

sudo mount -t auto /dev/sdb1 /mnt/earth

Un-Mounting

umount /dev/sdb1

Auto-Mounting

Create directory to mount to Add permissions if required Run

sudo blkid

to find the UUID and type Edit /etc/fstab Add entry like this, changing UUID and mount point

UUID=faba779f-6c49-46ea-85bf-c40371c80012  /mnt/earth  ext4

defaults 0 2

Run

sudo mount a

Run

sudo systemctl daemon-reload
